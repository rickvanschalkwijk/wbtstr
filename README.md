## Overview
WbTstr is a framework which makes testing the Web fun and productive. Under the hood, it uses Selenium WebDriver, which allows testing in all major browsers. We provide users with a very intuitive API that allows them to fully automate their functional testing without the hassle of complicated testing frameworks.

Let's get started!
